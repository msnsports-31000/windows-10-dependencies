<p>Note that this is still WIP, and arm dependencies arent complete yet.</p>
<p>This was made for Windows 10 UWP Applications that need such a dependency to run, overall, all the APPX files will be renamed to make the list more cleaner.</p>
<p>Q: What is a dependency?</p>
<p>A: A dependency refers to a piece of code, library, or software component that an application needs in order to run properly. Instead of writing every single function from scratch, developers rely on these pre-built packages</p>
